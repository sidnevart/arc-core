# Artifact Checklist

Use this checklist after each substantial MVP task.

## Always Check

- Is the implementation aligned with `plan.md`?
- Was the narrowest shippable slice chosen?
- Were risky actions held behind approval?
- Were claims verified by code, docs, or commands?

## Update When Needed

- `memory_bank/DECISIONS.md` for new stable project decisions
- `memory_bank/OPEN_QUESTIONS.md` for unresolved blockers
- `memory_bank/WORKLOG.md` for meaningful project progress
- `README.md` when operator-facing setup or workflow changed

## Verification Bar

- run the strongest local checks available
- note anything not verified
- call out residual risks explicitly
